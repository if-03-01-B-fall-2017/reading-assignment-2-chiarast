HTML

  1. Name the components of html elements properly: Headings(h1 - h6), Paragraphs(p), Links(a href = ...), Images
  2. Identify void elements and how they are denoted: void elements only have a start tage
  3. Identify attributes: Attributes usually come in name/value pairs like: name="value"
  4. Write down the correct DOCTYPE declaration for html 5: !DOCTYPE html
  5. Write down syntactically correct statements for the following elements:
    • Article (article): The article tag specifies independent, self-contained content
    • Body (body): The body element defines the document body
    • line break (br/): br is an empty element without a closing tag
    • Headings (h1, ...): h1 defines the most important heading. h6 defines the least important heading
    • Section with meta-information (head): The head element is a container for metadata (data about data) and is placed between the html tag and the body tag
    • Top level element (html): The html element defines the whole document
    • Image named Team.jpg (img src="Team.jpg"): HTML images are defined with the img tag. The source file (src), alternative text (alt), width, and height are provided as attributes
    • Paragraph (p): The HTML p element defines a paragraph. You cannot be sure how HTML will be displayed
    • Section (section): The section tag defines sections in a document, such as chapters, headers, footers, or any other sections of the document
    • Label appearing in the title bar (title): The title tag is required in all HTML documents and it defines the title of the document
    • Link to other pages (a): HTML links are defined with the a tag
    • Comment: You can add comments to your HTML source by using the following syntax:
    • Head with title and meta elements: The head element is a container for metadata (data about data) and is placed between the html tag and the body tag. The following tags describe metadata: title, style, meta, link, script, and base

CSS

  1. Name the components of a css rule properly: Selector, Declaration, Property, Value
  2. Use combinators properly: direct child, descendant, adjacent sibling, sibling:
  3. Specify colors by color names and in hexadecimal notation: Example: h1 style="color:Tomato">Hello World /h1
  4. Declarations for color, background color: body background-color: lightblue;
  5. Pseudo classes for the a element: A pseudo-class is used to define a special state of an element
  6. Declaration for background image: body background-image: url("paper.gif");
  7. Declaration for text alignment (left, right, center, justify): h1 text-align: left; rigth; center; justify;
  8. Declaration for text decoration (overline, underline, line-through): h1 text-decoration: overline; underline; line-through;
  9. Declaration for text transformation (uppercase, lowercase, capitalize): p.uppercase text-transform: uppercase; lowercase; capitalize;
  10. Identify the difference between serif and sans-serif fonts:
    • Serif fonts have small lines at the ends on some characters
    • "Sans" means without - these fonts do not have the lines at the ends of characters
  11. Declarations for font families: a specific font family (like "Times New Roman" or "Arial")
  12. Declarations for font style normal and italic: p.normal font-style: normal; italic;
  13. Declarations for font sizes: h1 font-size: 40px; or h1 font-size: 2.5em; /* 40px/16=2.5em */
  14. Declarations for font weights: p.normal font-weight: normal;
